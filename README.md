# Zsh setup for Iterm 2

### Install Iterm 2
https://www.iterm2.com

### Jump words
1. Go to `Preferences... > Profile > Keys`
2. Press `Load Preset`
3. Select `Natural Text Editing`

### 


### Install Zsh:
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
